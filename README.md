# Intent Classification: Few-Shot & Zero-Shot vs Fine-Tuning

Дипломная работа: сравнительный анализ подходов к классификации интентов на датасете CLINC150.

## Стек
- Python, Jupyter Notebook
- HuggingFace Transformers (DistilBERT, Flan-T5)
- scikit-learn

## Структура проекта
- `/data` — скрипты для загрузки и обработки данных
- `/experiments` — эксперименты: fine-tuning, zero-shot, few-shot
- `/notebooks` — Jupyter Notebooks для анализа и визуализации
- `/src` — общие утилиты и функции
- `/results` — метрики, графики, логи, модели
- `/docs` — описание проекта и инструкции
